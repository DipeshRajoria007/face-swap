FaceSwap Extension Privacy Policy

Effective date: February 10, 2026

Company: [Company Name]

Summary

FaceSwap is an internal productivity extension that swaps a user session on allowlisted sites by reading a token from the clipboard at the user’s request. We do not sell or share user data, and we do not use data for advertising or analytics.

Data We Access

  • Authentication information: The extension reads an auth token from the clipboard only when the user triggers a swap. The token is used immediately to set the target site’s localStorage and is not stored or transmitted by the extension.
  • Web activity (active tab URL): The extension reads the active tab URL to verify it is allowlisted and to clear the target site’s auth cookie as part of the swap.

How We Use Data

We use the clipboard token and active tab URL solely to perform the identity swap on the current allowlisted site after a user-initiated action.

Data Storage

The extension does not store user data. The token is written into the target site’s localStorage as part of the swap, and the target site’s auth cookie may be cleared. The extension itself does not persist or transmit any data.

Data Sharing

We do not sell, transfer, or share user data with third parties. No analytics or tracking services are used.

Third-Party Services

The extension does not execute remote code. The popup UI may load fonts from Google Fonts, which could receive standard web request metadata (such as IP address) from the browser.

Changes

We may update this policy from time to time. The effective date above will reflect the latest version.
